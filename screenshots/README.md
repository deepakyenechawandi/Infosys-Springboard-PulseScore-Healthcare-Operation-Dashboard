# Screenshots Directory

## How to Add Screenshots

1. **Run the application**: `streamlit run app.py`

2. **Capture screenshots** for each section:
   - `national_snapshot.png` - National Snapshot page
   - `structural_gaps.png` - Structural Gap Diagnosis page
   - `resource_distribution.png` - Resource Distribution page
   - `surge_intelligence.png` - Surge Risk Intelligence page
   - `hospital_finder.png` - Hospital Finder page

3. **Save screenshots** in this directory with the exact filenames listed above

4. **Recommended settings**:
   - Resolution: 1920x1080 or higher
   - Format: PNG
   - Capture full page or key sections
   - Use dark mode for consistent branding

## Tips

- Use browser screenshot tools or tools like Snagit, Greenshot, or Windows Snipping Tool
- Ensure sensitive data is not visible in screenshots
- Capture the most visually appealing sections with data loaded
- Consider adding annotations to highlight key features
